  * Why: To Fix the weaknesses Reported by Vulnerabilities Assessment or others.  
  * How: Enforce packages updates for both OS and Softwares.

  * Why: To Fix the weaknesses Reported by Vulnerabilities Assessment or CIS Benchmark or others.
